# Travel Flutter App

이 프로젝트는 iOS/Android 동시 지원 Flutter 앱입니다.

## 주요 기능
- 위치 기반 주변 여행지 정보 제공 (테마별 분류, 구글 플레이스 연동)
- 여행 계획 작성 및 캘린더 조회
- 여행지별 숙박업소 검색 (구글 플레이스 연동)
- AI 연동 여행 추천
- 하단 네비게이션 테마 적용

## 폴더 구조
- `lib/` : 주요 소스 코드
- `assets/` : 이미지 및 기타 리소스
- `README.md` : 프로젝트 설명

## 실행 방법
1. Flutter 환경이 설치되어 있어야 합니다.
2. `flutter pub get` 명령어로 의존성 설치
3. `flutter run`으로 앱 실행

추가 기능 및 디자인은 요구사항에 따라 확장 가능합니다.
